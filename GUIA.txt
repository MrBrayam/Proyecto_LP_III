1. REQUISITOS FUNCIONALES
1.1. MÓDULO I: GESTIÓN DE PLATAFORMA (SUPERADMIN)
Este módulo gestiona las funcionalidades centrales de administración de la plataforma SaaS orientada a negocios del sector belleza. Incluye la gestión de negocios registrados (tenants), planes de suscripción, facturación automática, configuraciones globales y herramientas de monitoreo.
RF-PLT-001: Registrar Nuevo Negocio (Tenant)
Define el proceso mediante el cual un nuevo negocio del rubro belleza se incorpora a la plataforma. El sistema debe permitir registrar información legal y comercial del negocio (razón social, RUC, dirección, datos de contacto, rubro específico, sedes iniciales), asignar un plan de suscripción inicial y generar automáticamente la estructura operativa necesaria, incluyendo el usuario administrador principal y configuraciones predeterminadas. El sistema debe garantizar aislamiento completo de datos entre negocios.
RF-PLT-002: Crear Plan de Suscripción
Permite al SuperAdmin definir los planes comerciales disponibles, configurando precios, periodicidad de pago (mensual, trimestral, anual), límites operativos y funcionalidades habilitadas según el plan.
RF-PLT-003: Editar Plan de Suscripción
Permite modificar características de planes existentes mediante un esquema de versionado, asegurando que las suscripciones activas mantengan sus condiciones vigentes mientras las nuevas contrataciones se realicen bajo configuraciones actualizadas.
RF-PLT-004: Desactivar Plan de Suscripción
Controla la descontinuación de planes comerciales, impidiendo nuevas suscripciones pero garantizando continuidad para negocios con suscripción activa. Registra la fecha de desactivación para auditoría.
RF-PLT-005: Listar y Comparar Planes de Suscripción
Proporciona herramientas visuales para listar, filtrar y comparar planes disponibles, mostrando precio, periodicidad, módulos incluidos, límites operativos y beneficios adicionales.
RF-PLT-006: Cambiar Plan de Suscripción
Permite actualizar o degradar el plan de suscripción de un negocio, validando límites operativos, calculando ajustes prorrateados y garantizando una transición segura sin pérdida de información.

1.1.2. Submódulo: Facturación y Ciclo de Pagos
RF-FAC-001: Facturar Suscripciones Automáticamente
Define el proceso de facturación recurrente: genera facturas automáticamente según el plan contratado y la periodicidad establecida, procesa pagos mediante pasarelas habilitadas, registra transacciones exitosas o fallidas y controla el estado de morosidad del negocio.
RF-FAC-002: Suspender o Cancelar Suscripción
Gestiona la suspensión temporal o cancelación definitiva de una suscripción, controlando el acceso del negocio, aplicando períodos de gracia, reteniendo datos históricos durante el tiempo definido y permitiendo condiciones para reactivación o cierre total del tenant.

1.1.3. Submódulo: Reportes y Monitoreo de Plataforma
RF-RPL-001: Generar Reportes Consolidados de la Plataforma
Permite al SuperAdmin generar reportes con métricas de crecimiento, ingresos recurrentes, retención de negocios, actividad operativa y uso de funcionalidades clave. Presenta dashboards ejecutivos con KPIs y permite exportación en PDF y Excel.

1.1.4. Submódulo: Configuración Global
RF-CGL-001: Configurar Parámetros Globales de la Plataforma
Centraliza la definición de parámetros globales (impuestos por defecto, monedas, métodos de pago, integraciones externas, políticas de suscripción). Cada cambio queda registrado con control de versiones y auditoría.

1.1.5. Submódulo: Auditoría y Seguridad
RF-CGL-002: Auditar Actividad de Usuarios y Sistema
Establece el registro obligatorio de todas las acciones críticas (creación de tenants, cambios de plan, modificaciones globales, accesos administrativos, intentos fallidos de autenticación, operaciones de facturación). Proporciona herramientas de consulta para detectar anomalías y garantizar trazabilidad completa.

1.2. MÓDULO II: ADMINISTRACIÓN DEL NEGOCIO
Este módulo permite la configuración estructural, operativa y de seguridad del sistema para negocios del sector belleza bajo un enfoque multi-tenant, asegurando que cada negocio opere de forma independiente y segura.
1.2.1. Submódulo: Configuración del Negocio y Sedes
RF-ADM-001: Configurar Datos del Negocio
Permite registrar, consultar y actualizar los datos generales del negocio (razón social, RUC, dirección fiscal, correo, contacto, nombre comercial, tipo de negocio). Cada negocio opera como una unidad independiente (tenant).
RF-ADM-002: Configurar Branding del Negocio
Permite personalizar la identidad visual (logotipo, colores corporativos, nombre visible, elementos gráficos). El branding se refleja en documentos, comprobantes y notificaciones enviadas a clientes.
RF-ADM-003: Registrar Nueva Sede
Permite registrar sedes físicas con información de dirección, distrito, teléfono, horarios de atención y responsable. Cada sede opera de manera independiente en cuanto a citas, ventas, inventario y caja.
RF-ADM-004: Editar Información de Sede
Permite actualizar información administrativa y operativa de una sede sin afectar la integridad del historial de ventas, citas, tratamientos, inventario, caja y auditoría.
RF-ADM-005: Desactivar Sede
Permite suspender temporalmente la operación de una sede sin eliminar su información histórica. La sede desactivada queda inhabilitada para nuevas operaciones pero conserva todos sus registros.
RF-ADM-006: Configurar Almacenes por Sede
Permite configurar uno o más almacenes asociados a una sede para gestionar el inventario de productos cosméticos y materiales de tratamientos, organizando el stock según categorías y condiciones de almacenamiento.
RF-ADM-007: Configurar Horarios de Operación por Sede
Permite definir los horarios de atención de cada sede (días laborales, apertura/cierre, horarios especiales o feriados), los cuales validan la disponibilidad de ventas, citas y reservas online.
RF-ADM-008: Configurar Restricciones para Tratamientos y Servicios
Permite configurar restricciones operativas para tratamientos: duración mínima/máxima, reglas de reserva anticipada, tiempos de descanso entre citas, disponibilidad de personal y condiciones por tipo de tratamiento.
RF-ADM-009: Configurar Zonas de Delivery
Permite definir zonas geográficas de reparto por sede (distritos, costos de envío, montos mínimos de compra, tiempos estimados de entrega) y validar automáticamente si una dirección pertenece a una zona habilitada.
RF-ADM-010: Activar/Desactivar Módulo de Citas y Agenda
Permite activar o desactivar la gestión de reservas de tratamientos por sede según el modelo operativo del negocio. Al desactivar, impide nuevas citas pero mantiene el historial de citas anteriores.

1.2.2. Submódulo: Gestión de Usuarios y Seguridad
RF-ADM-011: Registrar Nuevo Usuario
Permite registrar nuevos usuarios con datos personales, credenciales, rol inicial y sedes asignadas (cajeros, recepcionistas, cosmetólogas, estilistas, especialistas, administradores, gerentes).
RF-ADM-012: Editar Usuario
Permite actualizar información de un usuario (datos personales, roles, sedes, estado) sin afectar el historial de operaciones realizadas.
RF-ADM-013: Desactivar Usuario
Permite desactivar un usuario para bloquear su acceso inmediato sin eliminar su información ni historial de acciones. Clave para gestionar rotación de personal.
RF-ADM-014: Asignar Rol a Usuario
Permite asignar uno o más roles a un usuario para definir su nivel de acceso y responsabilidades. La asignación es flexible y permite combinar roles según la estructura operativa.
RF-ADM-015: Asignar Usuario a Sedes
Permite definir en qué sedes puede operar un usuario, restringiendo su acceso a la información correspondiente a la sede donde trabaja.
RF-ADM-016: Crear Rol Personalizado
Permite crear roles personalizados como "Especialista en Tratamientos", "Asistente de Estética", "Encargado de Almacén" o "Supervisor de Atención al Cliente".
RF-ADM-017: Asignar Permisos a Rol
Permite definir permisos funcionales por rol, controlando el acceso granular a módulos (crear, editar, eliminar o visualizar en productos, servicios, citas, clientes, ventas, inventario, reportes y finanzas).
RF-ADM-018: Consultar Log de Auditoría
Permite consultar el registro histórico de acciones críticas realizadas por usuarios (creación/edición de productos, ventas, cancelación de citas, ajustes de inventario, cambios de configuración).
RF-ADM-020: Autenticación de Usuario
Permite autenticar usuarios mediante validación de usuario y contraseña, control de sesiones activas y registro de intentos fallidos, garantizando acceso solo a funcionalidades autorizadas por rol y sede.
RF-ADM-021: Recuperar Contraseña
Permite al usuario recuperar su acceso mediante verificación por correo electrónico o código de seguridad, estableciendo una nueva clave que cumpla políticas mínimas de seguridad.

1.2.3. Submódulo: Configuración General
RF-ADM-022: Configurar Parámetros del Sistema
Permite al Administrador definir y actualizar parámetros globales: moneda, impuestos, límites de stock, políticas de seguridad, reglas de ventas, tiempos de atención y límites de cancelación. Los parámetros aplican a nivel de tenant y pueden ser heredados por las sedes.
RF-ADM-023: Configurar Preferencias de Usuario
Permite a cada usuario configurar preferencias personales (idioma, formato de fecha, zona horaria, notificaciones, vista inicial) mejorando la experiencia de uso y productividad.
RF-ADM-024: Gestionar (CRUD) Notificaciones
Permite gestionar notificaciones operativas y administrativas (stock bajo, productos próximos a vencer, citas, cancelaciones, accesos sospechosos, etc.), configurando canales de envío (correo, sistema interno, WhatsApp, SMS) y niveles de prioridad.
RF-ADM-025: Configurar Métodos de Pago
Permite definir y administrar los métodos de pago habilitados: efectivo, tarjeta de crédito/débito, transferencias, billeteras digitales (Yape, Plin), pago contra entrega y pagos mixtos, tanto para POS como para tienda online.

1.3. MÓDULO III: GESTIÓN DE PRODUCTOS, INVENTARIO, COMPRAS Y PROVEEDORES
Gestiona el catálogo completo de productos cosméticos e insumos para tratamientos, el control de inventario con sistema FIFO para productos perecederos, y las operaciones de compras con proveedores.
1.3.1. Submódulo: Catálogo de Productos
RF-PRO-001: Registrar Nuevo Producto
Permite registrar productos con información específica del sector: tipo (maquillaje, skincare, fragancias, cuidado capilar, uñas, depilación, accesorios), marca, presentación, contenido neto, tipo de piel recomendado, ingredientes, lote, fecha de vencimiento, precio de costo y venta, categoría y visibilidad en storefront.
RF-PRO-002: Editar Producto
Permite actualizar atributos de productos existentes (nombre, descripción, precios, marca, presentación, visibilidad) sin afectar el historial de movimientos de inventario ni ventas realizadas.
RF-PRO-003: Desactivar Producto
Permite desactivar temporalmente productos del catálogo sin eliminar su historial (inventario, compras, ventas, reportes). Esencial para productos descontinuados o retirados.
RF-PRO-004: Listar Productos
Proporciona una vista completa del catálogo con búsqueda, filtrado, paginación y ordenamiento, mostrando nombre, marca, categoría, stock, precio, estado y visibilidad en storefront.
RF-PRO-005: Cargar Imágenes de Producto
Permite subir múltiples imágenes por producto, definir la imagen principal, ordenarlas y agregar textos alternativos para accesibilidad y presentación en storefront.
RF-PRO-006: Crear Categoría de Productos
Permite crear categorías para clasificar productos (maquillaje, cuidado facial, corporal, capilar, perfumes, uñas, depilación, accesorios, productos dermatológicos, etc.), facilitando la navegación en el storefront y la generación de reportes segmentados.
RF-PRO-007: Editar Categoría de Productos
Permite modificar nombre, descripción, imagen representativa y orden de visualización de categorías existentes, sin afectar el historial de productos asociados.
RF-PRO-008: Listar Categorías de Productos
Muestra una lista completa de categorías con nombre, estado, cantidad de productos asociados y fecha de creación, con filtros y búsquedas rápidas.
RF-PRO-009: Desactivar Categoría
Permite desactivar categorías sin uso, manteniendo el historial de productos que la tuvieron asignada para garantizar integridad en reportes y auditoría.
RF-PRO-010: Crear Marca
Permite registrar marcas cosméticas con nombre, país de origen, logo, descripción comercial y línea de productos para clasificar el catálogo y generar reportes de ventas por marca.
RF-PRO-011: Editar Marca
Permite modificar nombre, país de origen, logo y descripción de marcas existentes, conservando el historial de productos asociados.
RF-PRO-012: Listar Marcas
Muestra una lista completa de marcas con nombre, país, estado y cantidad de productos asociados, con opciones de filtrado y búsqueda rápida.
RF-PRO-013: Desactivar Marca
Permite desactivar marcas que ya no se comercializan, sin afectar el historial de productos ni los reportes de ventas históricas y auditorías.
RF-PRO-014: Crear Combo Promocional
Permite crear packs promocionales que agrupan múltiples productos con precio especial (ej. "Kit Skincare Básico", "Pack Maquillaje Completo"). Al venderse, el sistema descompone automáticamente el combo y descuenta el inventario de cada artículo incluido.
RF-PRO-015: Editar Combo
Permite modificar información de combos existentes (precios, vigencia, nombre, descripción, visibilidad) sin afectar ventas históricas ni registros contables previos.
RF-PRO-016: Definir Composición de Combo
Permite definir qué productos y en qué cantidades componen cada combo, validando stock disponible y permitiendo modificaciones controladas.
RF-PRO-017: Desactivar Combo
Permite desactivar combos que ya no están vigentes, conservando el historial de ventas para reportes y auditoría.
RF-PRO-018: Configurar Visibilidad en Storefront
Permite controlar qué productos y combos son visibles en la tienda online, mediante etiquetas como "Nuevo", "Oferta", "Recomendado", "Top Ventas" o "Edición Limitada".
RF-PRO-019: Búsqueda de Productos
Proporciona búsqueda avanzada por nombre, código interno, código de barras, categoría, marca, tipo de producto, línea cosmética o presentación, optimizando la operación en POS y administración de inventario.
RF-PRO-020: Importar Productos Masivamente
Permite importar múltiples productos desde Excel o CSV, validando los datos, detectando inconsistencias y generando un reporte de errores para corrección.
RF-PRO-021: Crear Promoción
Permite crear promociones con descuentos automáticos aplicables a productos específicos o categorías completas durante un período definido (liquidación, temporada, clientes frecuentes).
RF-PRO-022: Configurar Reglas de Promoción
Permite definir reglas avanzadas: 2x1, 3x2, descuentos escalonados por volumen, por compra mínima o regalos al adquirir productos específicos. El sistema aplica el descuento automáticamente durante la venta en POS o storefront.

1.3.2. Submódulo: Control de Inventario
RF-INV-001: Visualizar Stock Consolidado por Producto
Proporciona una vista consolidada del inventario disponible por producto, sumando stock de todos los lotes activos en todos los almacenes, con alertas visuales para stock bajo o crítico.
RF-INV-002: Registrar Entrada de Lote de Inventario
Permite registrar la entrada de un nuevo lote capturando fecha de vencimiento, costo unitario, proveedor, número de lote del fabricante, fecha de ingreso y observaciones. Especialmente importante para cosméticos y dermatológicos con fechas de caducidad.
RF-INV-003: Aplicar Sistema FIFO para Descuento Automático de Inventario
Implementa el método FIFO (First In, First Out) para descontar automáticamente el inventario desde los lotes más antiguos o próximos a vencer primero, minimizando pérdidas por vencimiento en cremas, sérums, tintes y tratamientos dermatológicos.
RF-INV-004: Configurar y Recibir Alertas de Stock Bajo
Monitorea continuamente los niveles de inventario y genera alertas automáticas cuando un producto alcanza su nivel mínimo o crítico, visibles para el Administrador y responsables de almacén.
RF-INV-005: Generar Alertas de Productos Próximos a Vencer
Monitorea fechas de vencimiento de todos los lotes activos y genera alertas preventivas para implementar estrategias de salida (descuentos, promociones, liquidaciones o retiro anticipado).
RF-INV-006: Registrar Movimientos de Inventario (Ajustes Manuales)
Permite registrar movimientos manuales para ajustar discrepancias: pérdidas, productos dañados, mermas, muestras gratuitas, uso interno para tratamientos o reclasificaciones. Cada ajuste guarda fecha, responsable, motivo y observaciones.
RF-INV-007: Transferir Inventario Entre Almacenes
Facilita la transferencia de productos entre almacenes de una misma sede o entre sedes del mismo negocio, registrando cantidad, producto, lote, origen y destino para mantener trazabilidad completa.
RF-INV-008: Rastrear Trazabilidad Completa de Lotes
Proporciona trazabilidad completa desde el ingreso de un lote hasta su salida final (venta, uso en tratamientos, merma o devolución), incluyendo fecha, cliente o servicio asociado y usuario responsable.
RF-INV-009: Registrar Producto Vencido o Dañado
Permite registrar y retirar inmediatamente del inventario productos vencidos o dañados, documentando motivo, lote afectado, cantidad y responsable. Gestiona la disposición final (descarte, devolución a proveedor o retiro por control sanitario).

1.3.3. Submódulo: Compras y Proveedores
RF-COP-001: Crear Proveedor
Permite registrar proveedores de productos cosméticos e insumos estéticos con razón social, RUC, dirección, teléfono, correo, persona de contacto, términos comerciales y categorías de productos suministrados.
RF-COP-002: Editar Proveedor
Permite editar información de proveedores existentes (contacto, condiciones comerciales, líneas de productos, términos de pago) para mantener la base de proveedores actualizada.
RF-COP-003: Desactivar Proveedor
Permite desactivar proveedores con los que ya no se trabaja, evitando su selección en nuevas órdenes de compra pero manteniendo el historial completo de compras realizadas.
RF-COP-004: Crear Orden de Compra
Permite crear órdenes de compra formales especificando productos, cantidades, precios unitarios, condiciones de entrega y forma de pago. Sirve como documento vinculante y punto de partida para la recepción de mercancía.
RF-COP-005: Aprobar/Rechazar Órdenes de Compra
Implementa un flujo de aprobación para órdenes que superen ciertos umbrales de monto, permitiendo a gerentes o responsables revisar, aprobar o rechazar compras antes de enviarlas al proveedor.
RF-COP-006: Recibir Mercancía y Generar Entrada de Inventario
Facilita el proceso de recepción de mercancía, verificando cantidades recibidas contra lo solicitado, registrando discrepancias, lotes y fechas de vencimiento. Una vez confirmada la recepción, genera automáticamente las entradas de inventario correspondientes.
RF-COP-007: Crear Devolución a Proveedor
Permite registrar devoluciones de productos defectuosos, dañados, vencidos o no conformes, incluyendo proveedor, productos, cantidades, lote, motivo y fecha.
RF-COP-008: Editar Devolución a Proveedor
Permite modificar devoluciones registradas mientras no hayan sido finalizadas (corregir cantidades, productos o motivos antes de la confirmación oficial).
RF-COP-009: Listar Devoluciones a Proveedores
Permite visualizar todas las devoluciones registradas con filtros por proveedor, fecha, estado y motivo, para evaluar la calidad del abastecimiento y detectar proveedores con mayores incidencias.
RF-COP-010: Consultar Cuentas por Pagar
Proporciona una vista consolidada de todas las obligaciones pendientes con proveedores (facturas pendientes, montos, fechas de vencimiento y estados de pago) para gestionar el flujo de caja.
RF-COP-011: Registrar Compra Directa
Permite registrar compras realizadas sin una orden de compra previa (adquisiciones urgentes o de bajo monto), generando automáticamente la entrada de inventario y la cuenta por pagar si fue al crédito.
RF-COP-012: Registrar Pago a Proveedor
Permite registrar pagos totales o parciales a proveedores mediante múltiples formas de pago (efectivo, transferencia, tarjeta). Cada pago genera un comprobante interno de egreso con fecha, responsable y número de operación.

1.4. MÓDULO IV: OPERACIONES DE VENTA
Gestiona todas las operaciones comerciales, incluyendo venta de productos en el POS, gestión de citas y tratamientos, administración de pedidos online, control de caja, devoluciones, comprobantes y reportes diarios.
1.4.1. Submódulo: Punto de Venta (POS)
RF-VTA-001: Abrir y Cerrar Sesión de Caja
Apertura: registro de caja, monto inicial de efectivo (fondo de caja) y validación de permisos.
Cierre: registro de efectivo contado físicamente, cálculo automático de diferencias (sobrantes/faltantes), consolidación de ventas por tipo de pago, generación de reporte de cierre y bloqueo automático de nuevas ventas.
Validaciones: alertas por diferencias mayores a límites configurados y monitoreo de sesiones abiertas más de 12 horas.
RF-VTA-002: Registrar Venta en POS (Productos y Servicios de Belleza)
Búsqueda por nombre, código interno, código de barras o SKU. Selección de tratamientos (faciales, manicure, pedicure, depilación, masajes, tintes, etc.) con precio definido o editable según permisos. Validación en tiempo real de disponibilidad de stock. Registro de cantidades, descuentos, impuestos y total final.
RF-VTA-003: Procesar Múltiples Formas de Pago
Soporta pagos mixtos dentro de una misma venta: efectivo (Soles), tarjetas de crédito/débito (Visa, Mastercard), transferencias bancarias, billeteras digitales (Yape, Plin con generación de QR), crédito interno para clientes autorizados, y vales o Gift Cards.
RF-VTA-004: Emitir Comprobantes de Pago (Boleta/Factura)
Emisión de comprobantes electrónicos conforme a la normativa SUNAT, incluyendo boleta de venta (DNI opcional), factura electrónica (RUC obligatorio), notas de crédito/débito para anulaciones o correcciones, y código QR con hash de firma digital para validación electrónica. Envío automático al proveedor OSE.
RF-VTA-005: Aplicar Descuentos y Promociones Automáticas
Evalúa automáticamente el carrito para aplicar: descuentos por producto o servicio (fijo o porcentual), promociones por cantidad (3x2, segunda unidad al 50%), combos o paquetes, promociones por horario (Happy Hour), y validación para que el precio final no sea menor al costo permitido.
RF-VTA-006: Administrar Devoluciones y Reembolsos
Búsqueda de la venta original por ticket, comprobante o cliente. Opciones de resolución: reembolso al mismo método de pago, cambio de producto o emisión de nota de crédito. Actualización de inventario (reingreso o registro como merma). Registro obligatorio del motivo de devolución para auditoría.
RF-VTA-007: Suspender y Recuperar Ventas
Permite suspender una transacción en curso guardando el carrito con estado "suspendida" y un código de recuperación único, liberando la interfaz del POS para continuar con nuevas ventas.
RF-VTA-008: Generar Reportes de Ventas del Día
Reportes disponibles: cierre de caja detallado por sesión, ventas por cajero/usuario, ventas por producto/categoría, ventas por servicio/tratamiento, ventas por forma de pago (efectivo vs digital). Exportación en PDF y Excel.
RF-VTA-009: Integrar con Impresora Fiscal/Térmica
Impresión de tickets de venta, comprobantes, reportes de cierre (X y Z) y órdenes internas (fichas de tratamiento). Soporte para impresoras USB, Red y Bluetooth.
RF-VTA-010: Gestionar Caja Chica y Gastos Menores
Permite registrar salidas de dinero desde el POS para gastos operativos menores (movilidad, compra de insumos, limpieza, etc.) con registro de monto, categoría, descripción y captura de comprobante de respaldo. Descuento automático del efectivo disponible al cierre de caja.

1.4.2. Submódulo: Gestión de Citas y Servicios (Agenda de Tratamientos)
RF-VTA-011: Registrar Cita para Tratamiento o Servicio
Registro de cliente (nuevo o existente), selección del servicio (facial, manicure, pedicure, depilación, masajes, etc.), asignación de especialista, fecha, hora, duración estimada y generación de código de cita único.
RF-VTA-012: Registrar Servicios en una Cita (Atención de Tratamiento)
Permite agregar servicios y productos consumidos en una cita programada. Incluye selección de servicios con variantes, registro de observaciones del especialista, posibilidad de agregar productos vendidos durante la atención y validación automática de stock si se usan insumos del inventario.
RF-VTA-013: Reprogramar o Transferir Citas
Permite reprogramar citas por fecha/hora, transferirlas a otro especialista disponible o a otra sede, registrando el motivo del cambio y validando disponibilidad antes de confirmar.
RF-VTA-014: Finalizar Cita y Generar Cobro
Genera el cobro al finalizar el servicio: resumen de servicios realizados, aplicación de descuentos o promociones, registro de método de pago, emisión de comprobante y cierre automático de la cita liberando el horario en la agenda.

1.4.3. Submódulo: Gestión de Pedidos (Productos y Reservas Online)
RF-VTA-015: Crear Pedido con Múltiples Modalidades
Soporta delivery (con dirección, cálculo de envío y asignación de repartidor), pickup (recojo en tienda con horario programado) y compra presencial, con validación automática de cobertura de zona para delivery.
RF-VTA-016: Confirmar y Procesar Pedidos
Validación de stock, confirmación de recepción, estimación de tiempo de entrega/recojo, notificación automática al cliente (WhatsApp/SMS/Correo) y asignación automática o manual de repartidor.
RF-VTA-017: Rastrear Estado de Pedido en Tiempo Real
Estados: Pendiente → Confirmado → En Preparación → Listo → En Camino → Entregado → Cancelado. Vista tipo Kanban para administración interna y link público de seguimiento para el cliente.
RF-VTA-018: Configurar Zonas y Costos de Delivery
Definición de zonas por polígonos, distritos o radio (km), configuración de tarifas fijas o por distancia, monto mínimo de compra, y validación automática de direcciones al crear pedidos.
RF-VTA-019: Asignar y Gestionar Repartidores
Registro de repartidores (datos personales, vehículo, placa), módulo de gestión para recepción y confirmación de entrega, y monitoreo de pedidos asignados en tiempo real.
RF-VTA-020: Gestionar Cancelaciones y Devoluciones de Pedidos
Cancelación por cliente o negocio con registro de motivo, gestión automática de reembolsos según políticas internas, y actualización de inventario en caso de cancelación antes de entrega.
RF-VTA-021: Reportes y Analítica de Pedidos
Análisis de volumen de pedidos por canal y modalidad, tiempo promedio de preparación y entrega, mapa de calor de zonas de delivery, y análisis de rentabilidad y costos logísticos.

1.4.4. Submódulo: Gestión de Clientes
RF-VTA-022: Administrar Quejas y Reclamos
Registro multicanal de incidencias (presencial, web, teléfono, WhatsApp). Flujo de resolución con asignación de responsables y SLA. Historial de soluciones aplicadas y registro de estado del reclamo (pendiente, en proceso, resuelto).
RF-VTA-023: Gestionar Clientes
Administración completa (CRUD) de clientes con nombres, DNI, teléfono, correo, dirección, historial de compras y tratamientos realizados. Facilita estrategias de fidelización y seguimiento del historial estético de cada cliente.

1.5. MÓDULO V: FACTURACIÓN Y FINANZAS
Sistema integral para la gestión fiscal, contable y financiera del negocio multi-tenant de belleza, garantizando cumplimiento normativo con SUNAT y control de ingresos y gastos operativos.
1.5.1. Submódulo: Facturación Electrónica
RF-FIN-001: Gestionar Series de Comprobantes Electrónicos
Administración de series por tipo de comprobante (boleta, factura, nota de crédito, nota de débito) y punto de emisión, con numeración correlativa única por sede, garantizando continuidad numérica y cumplimiento normativo.
RF-FIN-002: Gestionar Emisión de Comprobantes Electrónicos
Emisión de boletas y facturas electrónicas con registro de datos del cliente, detalle de productos/servicios, impuestos, descuentos, totales y método de pago. Valida información fiscal obligatoria (DNI o RUC).
RF-FIN-003: Enviar Comprobantes a SUNAT
Envío automático a SUNAT mediante servicio OSE, gestionando respuestas de aceptación, observación o rechazo, y manteniendo historial de envíos con fecha, hora y mensajes de respuesta.
RF-FIN-004: Consultar y Controlar Estado de Comprobantes
Consulta del estado de comprobantes enviados: aceptados, observados, rechazados o pendientes de envío. Permite filtros por fecha, tipo, cliente o sede para detectar errores y aplicar correcciones.
RF-FIN-005: Gestionar Notas de Crédito y Débito
Emisión de notas de crédito y débito asociadas a comprobantes previos para corregir errores, anular parcialmente operaciones o ajustar montos. Mantiene coherencia contable y tributaria con registro del motivo según códigos SUNAT.
RF-FIN-006: Gestionar Anulación de Comprobantes
Controla la anulación de comprobantes dentro de los plazos permitidos por SUNAT, registrando motivo, generando ajustes financieros, actualizando estados en inventario y ventas, y manteniendo trazabilidad para auditoría.
RF-FIN-007: Gestionar Reimpresión y Consulta de Comprobantes
Permite consultar y reimprimir comprobantes emitidos sin alterar la información original, garantizando integridad documental y facilitando copias para reclamos, devoluciones o requerimientos contables.

1.5.2. Submódulo: Devoluciones y Ajustes
RF-FIN-008: Registrar Solicitudes de Devolución
Permite registrar devoluciones de productos cosméticos o servicios cobrados incorrectamente, asociadas siempre a la venta original, con control de cantidades devueltas, montos, fecha de solicitud y motivo.
RF-FIN-009: Gestionar Motivos y Aprobación de Devoluciones
Clasificación de devoluciones por motivos predefinidos (producto defectuoso, error de facturación, producto vencido, insatisfacción del cliente, servicio no realizado) con proceso de aprobación por personal con permisos adecuados.
RF-FIN-010: Generar Ajustes Financieros por Devolución
Generación automática de ajustes financieros tras una devolución: notas de crédito, reversos contables o actualizaciones de ingresos, vinculados al comprobante original y reflejados en reportes financieros.
RF-FIN-011: Gestionar Reembolsos al Cliente
Gestión de la devolución del dinero al cliente (efectivo, transferencia, devolución a tarjeta o billetera digital) con registro del monto, fecha, responsable y estado de la operación para trazabilidad financiera.
RF-FIN-012: Reintegrar Productos Devueltos al Inventario
Reincorpora al inventario los productos devueltos en condiciones aptas, actualizando niveles de stock y registrando el movimiento como entrada por devolución. Productos dañados o abiertos se registran como merma o descarte.

1.5.3. Submódulo: Gestión de Gastos
RF-FIN-013: Registrar Gastos Operativos
Permite registrar gastos del negocio (insumos, alquiler, servicios básicos, pago de personal, mantenimiento, materiales de limpieza) con monto, fecha, concepto, categoría y proveedor.
RF-FIN-014: Clasificar y Documentar Gastos
Categorización de gastos (insumos estéticos, cosméticos, publicidad, servicios básicos, mantenimiento, transporte, personal) con posibilidad de adjuntar evidencia digital del gasto (foto o archivo escaneado).
RF-FIN-015: Aprobar Gastos Operativos
Proceso de validación y aprobación de gastos por un administrador o gerente antes de su contabilización final, controlando el uso de recursos económicos y previniendo registros incorrectos o no autorizados.
RF-FIN-016: Gestionar Gastos Recurrentes
Configuración de gastos periódicos (alquiler, internet, luz, agua, suscripciones, mantenimiento de equipos, compras programadas de insumos) con registro automatizado según la frecuencia definida (mensual, semanal, anual) y alertas de vencimiento.

1.6. MÓDULO VI: E-COMMERCE Y REPORTES
Plataforma de comercio electrónico integrada con el sistema POS para ventas online de productos cosméticos, gestión de pedidos web y motor avanzado de reportes con dashboards estratégicos.
1.6.1. Submódulo: Configuración de Storefront (Admin)
RF-ECO-001: Configurar Tienda Online
Permite configurar la tienda online con información general del negocio, políticas comerciales, condiciones de compra, métodos de pago, opciones de entrega, redes sociales, información de contacto y políticas de cambios/devoluciones, integrada en tiempo real con inventario y POS.
RF-ECO-002: Configurar Métodos de Pago Online
Habilita y gestiona métodos de pago online: tarjetas de crédito/débito, transferencias bancarias, Yape/Plin con QR, PagoEfectivo u otros medios digitales, billeteras digitales integradas, y pago contra entrega para pedidos delivery.
RF-ECO-003: Configurar Métodos de Entrega
Define y administra modalidades de entrega: delivery a domicilio (con zonas, costos y tiempos estimados) y recojo en tienda (con sedes disponibles y horarios de atención), con reglas de disponibilidad por sede.
RF-ECO-004: Configurar Horarios de Venta Online
Permite configurar los horarios en que la tienda online recibe pedidos, por día, con restricciones en feriados o fechas especiales, para gestionar la capacidad operativa y evitar pedidos fuera de horario.
RF-ECO-005: Activar/Desactivar Tienda Online
Permite activar, desactivar temporalmente o colocar en modo mantenimiento la tienda online, mostrando un mensaje personalizado a los clientes cuando esté suspendida.

1.6.2. Submódulo: Tienda Online (Cliente)
RF-ECO-006: Navegar Catálogo de Productos
Permite explorar el catálogo completo de productos cosméticos con nombre, precio, imágenes, categoría, marca y disponibilidad, organizado por secciones (maquillaje, skincare, cuidado capilar, perfumes, uñas, accesorios).
RF-ECO-007: Filtrar Productos por Categoría / Marca / Precio
Ofrece filtros avanzados por categorías, marcas, rango de precios y disponibilidad para encontrar rápidamente productos de interés.
RF-ECO-008: Buscar Productos
Permite búsquedas textuales por nombre, descripción, marca, tipo de producto o características específicas (ej. "base mate", "crema antiarrugas", "shampoo keratina", "protector solar").
RF-ECO-009: Ver Detalle de Producto
Muestra página de detalle con imágenes, descripción, precio, stock disponible, marca, presentación, recomendaciones de uso, tipo de piel/cabello recomendado, advertencias, condiciones de conservación y productos relacionados para cross-selling.
RF-ECO-010: Agregar Producto al Carrito
Permite agregar productos al carrito con validación de stock en tiempo real, verificando que el producto esté habilitado para venta online y permitiendo agregar múltiples productos y modificar cantidades.
RF-ECO-011: Modificar Carrito de Compras
Permite modificar el carrito antes del checkout (aumentar/reducir cantidades, eliminar productos, vaciar el carrito) con recálculo automático de subtotales, descuentos, impuestos y monto final.
RF-ECO-012: Seleccionar Modalidad de Entrega
Permite seleccionar delivery a domicilio (con cálculo automático del costo por zona y tiempo estimado) o recojo en tienda (seleccionando sede y horario disponible).
RF-ECO-013: Procesar Pago Online
Integración con pasarelas de pago para procesar transacciones online de manera segura, registrando intentos de pago, confirmaciones, pagos por transferencias o billeteras digitales, y generando conciliación financiera correcta.
RF-ECO-014: Rastrear Pedido
Sistema de seguimiento del estado del pedido en tiempo real: Pendiente de pago → Pago verificado → Confirmado → En preparación → Listo para envío → En camino → Listo para recojo → Entregado → Cancelado.
RF-ECO-015: Registro y Login de Cliente
Permite a los clientes crear una cuenta (registro, login, recuperación de contraseña, gestión de perfil) almacenando datos personales, historial de compras, direcciones frecuentes y preferencias para fidelización y campañas personalizadas.

1.6.3. Submódulo: Reportes y Análisis
RF-ECO-016: Reporte de Ventas
Análisis integral de ventas por día, semana y mes, por sede, comparación entre sedes, identificación de la sede con mayores ingresos, y segmentación entre ventas de productos y servicios.
RF-ECO-017: Reporte de Productos Más Vendidos
Análisis de productos top por volumen de ventas (best sellers, alta rotación, baja demanda) con filtros por sede, categoría, marca y período. Soporte para estrategias de marketing y optimización del catálogo.
RF-ECO-018: Reporte de Servicios y Tratamientos Más Solicitados
Reporte de frecuencia de citas, ingresos por servicio, duración promedio y especialistas con mayor demanda, para optimizar la agenda, planificar personal y diseñar promociones estratégicas.
RF-ECO-019: Reporte de Inventario
Estado del inventario con stock disponible, stock bajo, stock crítico, productos próximos a vencer y margen de ganancia por producto (precio de costo vs precio de venta).
RF-ECO-020: Reporte de Movimientos de Inventario
Trazabilidad completa de movimientos (entradas, salidas, ajustes, transferencias, devoluciones, mermas) con fecha, usuario responsable, motivo y lote involucrado.
RF-ECO-021: Reporte de Compras y Gastos
Consolidado de compras a proveedores y gastos operativos, analizable por categoría, proveedor y período (insumos, cosméticos, mantenimiento, publicidad, servicios básicos y otros costos operativos).
RF-ECO-022: Reporte de Boletas Emitidas
Reporte consolidado de boletas de venta por período: número de comprobante, fecha, cliente, monto total, sede y estado SUNAT. Incluye identificación de horarios pico de compra y productos más vendidos.
RF-ECO-023: Reporte de Facturas Emitidas
Reporte detallado de facturas emitidas con análisis de montos, estados, clientes corporativos más relevantes, tendencias de facturación y comprobantes pendientes o con observaciones.
RF-ECO-024: Exportar Reportes (Excel, PDF)
Permite exportar cualquier reporte en Excel o PDF, conservando la estructura del reporte y los filtros aplicados, para presentaciones ante gerencia, contabilidad o auditorías.

2. REQUISITOS NO FUNCIONALES
2.1. REQUISITOS DE SEGURIDAD
Toda la comunicación entre cliente y servidor debe realizarse mediante HTTPS, cifrando datos sensibles (credenciales, información de clientes, facturación, transacciones POS y tienda online). Las contraseñas deben almacenarse con hash robusto (bcrypt), sin texto plano. El sistema implementa control de acceso basado en roles (RBAC) y maneja sesiones mediante tokens únicos con expiración automática por inactividad. Bajo arquitectura multi-tenant, cada negocio debe estar completamente aislado y toda operación debe validar el tenant correspondiente antes de procesar datos.
2.2. REQUISITOS DE ESCALABILIDAD (MULTI-TENANT)
La arquitectura multi-tenant debe permitir que múltiples negocios de belleza utilicen la plataforma simultáneamente de manera independiente y segura, identificados por un ID único de tenant validado en todas las operaciones. El sistema debe soportar el crecimiento progresivo de la plataforma (nuevos negocios, múltiples sedes, incremento de ventas, citas y pedidos online) sin afectar el rendimiento general. Cada tenant debe contar con su propia configuración de sedes, usuarios, catálogo, inventario, clientes, facturación y reportes.
2.3. REQUISITOS DE DISPONIBILIDAD
El sistema debe estar disponible en horarios comerciales y extendidos, especialmente en módulos críticos (POS, agenda de citas, inventario, facturación). Debe contar con mecanismos de respaldo de información y recuperación ante fallos, evitando pérdidas de datos críticos (ventas, historial de clientes, citas, movimientos de inventario). Asimismo, debe manejar correctamente interrupciones temporales de conexión permitiendo retomar operaciones sin inconsistencias una vez restablecido el servicio.
2.4. REQUISITOS DE USABILIDAD
La interfaz debe ser responsive (computadoras, tablets y dispositivos móviles) con navegación intuitiva, formularios simplificados, botones visibles y procesos guiados para tareas frecuentes (registrar ventas, programar citas, registrar tratamientos, consultar inventario, generar comprobantes). El sistema debe mostrar mensajes claros de error o confirmación, y el personal diverso (cajeros, recepcionistas, estilistas, cosmetólogas, administradores) debe poder operarlo con capacitación mínima.
2.5. REQUISITOS DE MANTENIBILIDAD
El código fuente debe estar organizado en capas estructuradas (controladores, servicios, repositorios) siguiendo arquitectura limpia o patrón MVC, facilitando la identificación y modificación de funcionalidades. El sistema debe estar preparado para futuras mejoras: nuevas pasarelas de pago, módulos de fidelización, reportes avanzados, automatización de campañas de marketing e incorporación de nuevos servicios estéticos, pudiendo adaptarse al crecimiento del negocio y a cambios normativos sin necesidad de reconstruir la plataforma.
